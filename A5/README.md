# Arborea

## Prerequisites

- Docker

### How to run with Docker

1. Clone the repository

```bash
git clone https://github.com/Agostax0/HCI-course
```

2. Move to the Assignment 5 directory

```bash
cd A5/
```

3. Run the docker container

```bash
docker compose up
```

4. The web-server will now be available [here](http://localhost:5173)


### How to run without Docker

1. Clone the repository

```bash
git clone https://github.com/Agostax0/HCI-course
```

2. Move to the arborea directory

```bash
cd A5/arborea/
```

3. Run the npm commands

```bash
npm install && npm run dev
```

4. The web-server will now be available [here](http://localhost:5173)

## Recommended Viewing Experience

It is highly advised to open the website with an iPhone16 layout (width=393, height=852, device pixel ratio = 3).
